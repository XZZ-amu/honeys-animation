# 规则 03：选项必须覆盖"物体的结构形态"，不只是"运动方式"

## 什么错
风铃的选项给了"悬挂摆动型/碰撞叮响型/串珠飘荡型"——都在描述运动方式，但没有一个描述了用户想要的物体结构（圆环+飘带）。用户被迫选了"最接近的"但其实都不对。

## 为什么错
选项的区分维度选错了。对于一个具体物体，用户脑中的第一层画面是"它长什么样"（结构），不是"它怎么动"（运动）。先对齐结构，再讨论运动。

## 怎么防

**选项设计规则**：

1. 第一轮选项应该区分**物体的结构/形态**（长什么样）
2. 结构确认后，再用第二轮（如果需要）区分运动方式或情绪

**示例**：
- ❌ "悬挂摆动型 / 碰撞叮响型 / 串珠飘荡型"（都在说运动）
- ✅ "横杆挂铃铛型 / 圆环挂飘带型 / 日式单管风铃型"（在说结构）

**如果不确定物体有哪些常见形态**：先查/想一下这个物体在现实中有哪几种典型样子，再设计选项。
