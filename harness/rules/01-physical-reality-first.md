# 规则 01：物理参数必须先锚定现实

## 什么错
风铃动画把"灵动俏皮"翻译成大幅度摆动（30-60度），实际真实风铃在微风下只有 5-10 度的轻微晃动。

## 为什么错
从情绪关键词直接推导物理参数。"俏皮 = 动得多"是错误的等式。物理参数的真理来源是现实世界，不是情绪。

## 怎么防

**强制检查点**：当动画涉及现实世界中存在的物体时，执行前必须先回答以下问题：

1. 这个物体在真实世界中是怎么动的？（幅度、速度、力度的真实范围）
2. 是什么力在驱动它？（微风？重力？弹力？）这个力在现实中有多大？
3. 情绪/氛围只影响视觉表现（颜色、节奏感、细节丰富度），不影响物理参数

**执行规则**：
- 物理参数（幅度、速度、力度）→ 从现实世界推导
- 视觉参数（颜色、风格、装饰）→ 从情绪推导
- 两者独立，不互相污染
