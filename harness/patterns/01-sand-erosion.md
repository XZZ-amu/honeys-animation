# 成功模式：沙粒侵蚀效果

## 适用场景
文字/形状需要"碎裂消散"效果——像沙雕被风蚀、像粉笔字被擦掉、像灰尘被吹走。

## 正确的心智模型
"元素本身就是由微小颗粒组成的，外力把颗粒逐渐带走，元素因此变得残缺直到消失。"

不是："元素飞走的时候从身上喷射粒子。"

## 实现方式

1. **初始化**：用离屏 canvas 渲染文字 → 扫描像素点（每 2px 采样）→ 生成沙粒数组
2. **静止渲染**：用 1px fillRect 画所有沙粒，远看像正常文字
3. **侵蚀逻辑**：
   - 每颗沙粒有 releaseProgress（基于 x 偏移 + 随机扰动）
   - 迎风侧（左边）先脱落
   - 脱落后顺风方向快速飞走（200-400px/s），0.2-0.4s 消失
   - 未脱落的沙粒继续跟着整体移动
4. **结果**：字母从完整 → 左边残缺 → 越来越少 → 消失

## 推荐参数
- 沙粒大小：1px（fillRect）
- 采样间距：每 2px 一个点
- 每字母约 30-80 颗沙粒
- 脱落后飞行速度：200-400 px/s
- 脱落后存活时间：0.2-0.4s
- 脱落顺序：左 → 右（迎风侧优先）

## 验证过的案例
- animations/20-sand-erosion-v2.html

## 注意事项
- 性能：全文 300 字母 × 50 粒 = 15000 个点，用 fillRect 不用 arc
- 如果卡顿，降低采样密度（每 3px）
- 沙粒太大（>2px）会看起来像"豆子"不像"沙"
